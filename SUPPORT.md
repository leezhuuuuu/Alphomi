# Support

## Where to Ask for What

- Bug reports: open a GitHub issue with reproduction steps
- Feature requests: open a GitHub issue using the feature template
- Architecture or contributor workflow questions: open an issue or discussion if enabled
- Security concerns: use the private route described in [SECURITY.md](SECURITY.md)

## Before Opening an Issue

- Read [README.md](README.md)
- Read [docs/guides/development.md](docs/guides/development.md)
- Run `pnpm typecheck`, `pnpm test`, and `pnpm smoke` when relevant
- Include logs, platform details, and the exact command you ran

## Support Expectations

Alphomi is an open-source project, not a guaranteed support service. Response times will vary with maintainer availability, but clear reports with reproduction steps are much easier to triage.
