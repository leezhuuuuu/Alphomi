# 支持说明

[English](SUPPORT.md) | 简体中文

## 什么问题去哪里提

- Bug 报告：在 GitHub 上提交 issue，并附上复现步骤
- 功能建议：使用 feature request 模板提交 issue
- 架构问题或贡献者工作流问题：提交 issue，或在启用 discussion 后发起讨论
- 安全问题：走 [SECURITY.zh-CN.md](SECURITY.zh-CN.md) 中的私下报告流程

## 提 issue 之前

- 阅读 [README.zh-CN.md](README.zh-CN.md)
- 阅读 [开发指南](docs/guides/development.zh-CN.md)
- 在适用时先运行 `pnpm typecheck`、`pnpm test` 和 `pnpm smoke`
- 提供日志、平台信息，以及你实际运行的完整命令

## 支持预期

Alphomi 是一个开源项目，而不是承诺 SLA 的支持服务。响应速度会随维护者时间安排而变化，但描述清晰、可复现的问题会更容易被处理。
