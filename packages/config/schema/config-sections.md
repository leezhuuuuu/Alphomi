# Alphomi Config Sections

## `driver`

Driver service, browser execution, vision calls, snapshots, and desktop bridge settings.

## `brain`

Brain service, LLM integration, workflow limits, streaming behavior, and trace settings.

## `desktop`

Desktop shell, renderer development URL, and UI defaults.

## `user_data`

Cookie and localStorage persistence settings.

See `docs/guides/configuration.md` for the contributor-facing configuration guide.
