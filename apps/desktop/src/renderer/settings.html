Settings

设置

这里只保留真正会生效的选项。当前版本可以在这里管理外观、新建标签页地址、LLM provider，以及工具可用性。

外观模式

选择浅色、深色，或跟随系统外观自动切换。

新建标签页地址

每次新建标签页时，浏览器都会先打开这里设置的地址。

支持直接输入域名，保存时会自动补全为 `https://`。也支持 `about:blank`。

LLM Provider

在这里管理用户级的模型 provider profile。新的对话回合会自动读取最新设置；如果环境变量或 `config.yaml` 覆盖了某些字段，下面也会明确标出来。

存储方式
Base URL、模型名和 endpoint mode 会保存在本地设置里；API key 会单独保存在本机私有存储中。保存后，Desktop 和 Brain 都会读取同一份生效配置。

工具开关

每个工具都可以单独启用或禁用。禁用后，它会从可用工具 schema 和系统提示词中收敛掉，并在真正执行时被拒绝。

生效方式
设置会持久化保存，并同步给 Desktop、Driver、Brain 三层。新的回合会立刻感知这些变化；运行中的回合如果尝试调用已禁用工具，也会被安全拦截。
